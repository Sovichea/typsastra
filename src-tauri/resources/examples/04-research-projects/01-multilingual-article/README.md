# Multilingual article project

This multi-file project keeps `typsastra:document-scripts` in `main.typ`, so
the configured languages also apply while editing included sections.

Latin uses English, Khmer uses Khmer, and Arabic uses Arabic. Install the
optional Arabic provider to activate its tools. Change the Latin assignment to
French or Spanish when reviewing that language; Typsastra does not merge
same-script dictionaries.

Tutorial: <https://github.com/Sovichea/typsastra/blob/main/docs/tutorials/MULTILINGUAL_SPELLCHECK.md>
